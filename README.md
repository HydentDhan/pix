Pixelmon Raid ModHey there! Welcome to the Pixelmon Raid Mod. I built this because I wanted real, massive-scale boss battles for Pixelmon, not just a slightly bigger Pokémon that dies in two hits.This mod creates global, server-wide Raid Boss events where the entire server has to team up to whittle down a shared, massive HP pool. It comes fully packed with a token economy, customizable loot tables, dynamic difficulty scaling, and a fully functional UI.🌟 FeaturesGlobal Shared HP: When the boss spawns as a giant statue in the overworld, everyone fights a "copy" of it. The damage players do inside their individual battles actively drains the global Boss HP bar at the top of the screen.Dynamic Difficulty: The boss gets harder or easier depending on how many players join the fight. Plus, if the server wins consecutive raids, the difficulty tier (1 through 5) automatically levels up, giving the boss even more HP!Built-in Token Economy & Shop: Players earn tokens for participating. Admins can set up an entirely custom in-game shop using a slick GUI where players can spend those tokens on items or command rewards.Advanced Loot Tables: Reward players based on their performance! You can configure specific drops for the Top 3 damage dealers, participation rewards for everyone else, and a special bounty for the player who gets the Killshot.Holographic Leaderboards: Spawns a floating, live-updating hologram in your world showing the top damage dealers.Discord Integration: Automatically sends a webhook to your Discord server when a boss spawns, announcing its name, difficulty, HP, and time limit.Anti-Cheese Mechanics: * OHKO moves (Fissure, Sheer Cold, etc.) automatically miss.Dynamaxing and Mega Evolutions are suppressed by the boss's aura.You cannot catch the Raid Boss; Pokéballs will bounce right off.The boss is spawned at Level 100 with perfect 31 IVs and 252 EVs in every stat.🎮 Player CommandsThese are the standard commands for your players to interact with the raid system./raid status - Shows the current active boss, its remaining HP, the phase, and how much time is left./raid warp - Teleports the player directly to the Raid Arena (if admins have set it up)./raid join - Pulls the player into the battle! You must be within 30 blocks of the boss statue to use this./raid shop - Opens the custom Token Shop GUI to spend your hard-earned Raid Tokens./raid tokens balance - Checks how many Raid Tokens you currently have./raid leaderboard - Opens a GUI showing the top damage dealers from the last raid./raidstats - Opens a massive GUI showing the all-time server leaderboard (Total Damage, Kills, Raids Joined).⚙️ Admin CommandsNote: All admin commands require Permission Level 2. /raidadmin - Opens the Master Admin Control Panel GUI. From here, you can toggle auto-raids, change difficulty, edit loot tables, and edit the shop visually!/raidadmin start - Bypasses the timer and forcefully spawns a boss right now./raidadmin stop - Instantly cancels the current raid, despawns the boss, and resets the arena./raidadmin forcewin - Instantly kills the boss and distributes the rewards as a victory./raidadmin setcenter - Sets the exact block where the giant Boss Statue will spawn./raidadmin setwarp - Sets the location where players will teleport when they type /raid warp./raidadmin setholo - Moves the floating holographic leaderboard to exactly where you are standing./raidadmin toggleholo - Turns the holographic leaderboard on or off./raidadmin tokens give <player> <amount> - Gives free Raid Tokens to a player./raidadmin tokens take <player> <amount> - Removes Raid Tokens from a player./raidadmin shop add <price> - Hold an item in your hand and type this command to add it to the Token Shop for the specified price./raidadmin reload - Reloads pixelmonraid.json so you can apply config changes without restarting the server.📂 Configuration (pixelmonraid.json)Almost everything in this mod can be tweaked in the config file.Customize exactly which Pokémon can spawn for Difficulty Tiers 1 through 5.Change the base HP of the boss for each tier (e.g., Tier 1 has 20,000 HP, Tier 5 has 120,000 HP).Enable or disable sounds, configure custom chat messages, change the UI glass pane colors, and set up your Discord webhook URL
